ISTRUZIONI PER L'ESECUZIONE DEL PROGRAMMA

Prima di lanciare il programma bisogna:
- Aggiornare nella classe Main il percorso del file "matrix"
- Scrivere sul file "matrix" che si trova nel percorso la matrice iniziale relativa al problema
- Scrivere i bounds delle varie variabili nella classe "Main"
Dopo aver effettuato questi due passi, l
- Spostarsi da terminale nel percorso contenente il file jar
- Lanciare il programma tramite il comando "java -jar newSimplex.jar"
